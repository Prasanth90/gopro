# Go Pro Hero 4 Session

## Camera:
  - Set the Camera in the APP mode

## Server:
  - Install python 3.6
  - Set pip in the Environment variable
  - cd server
  - pip install -r requirements.txt
  - python gopro.py

## Client:
  - cd client
  - Run Mongoose server (just execute it)
